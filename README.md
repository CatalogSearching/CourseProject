# Система полнотекстового поиска

### Сборка программы:
**Первая сборка:**
```````
make
```````
**Последующие:**
```````
make rebuild
```````
### Сборка тестов:
```````
make test
```````
### Запуск программы:
```````
./bin/search <строка> <директория>
```````
### Запуск тестов:
```````
./bin/test
```````
